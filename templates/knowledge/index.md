# Knowledge index

> Topic -> file map. Before stating facts, use this map to find the right file, then grep / read it.
>
> This `knowledge/` dir is a small persistent wiki (after Karpathy's LLM-wiki idea): `index.md`
> is the catalog, `log.md` its chronological companion. Pages may carry associative cross-links
> `[[topic]]` (resolving to `knowledge/<topic>.md`) so related takes are one hop apart — add a
> "see also [[other-topic]]" line where a claim leans on another page.
