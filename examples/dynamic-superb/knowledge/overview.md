# What Dynamic-SUPERB is

A dynamic, collaborative benchmark for **universal speech models** that use
**instruction tuning** to perform multiple tasks in a **zero-shot** fashion — a
platform to evaluate and compare models across speech-processing tasks. [src:r1]

Seed scale: **55 evaluation instances** from **33 tasks × 22 datasets**, covering
content, speaker, semantics, degradation, and paralinguistics, plus audio-processing
tasks beyond speech. Each task has one or more datasets (instances) with labeled
examples and corresponding instructions. [src:r1]

It is community-driven: new tasks are submitted, reviewed for quality/feasibility/
relevance, then merged; model scores are posted to a leaderboard. Paper:
arXiv:2309.09510. (Submission/review/leaderboard docs are referenced in the README
but were not sampled.) [src:r1]
